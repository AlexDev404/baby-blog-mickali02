Week 3

For Self-Assessment #1, we completed a basic Go web app project aimed at enhancing our understanding of Go and Git. The task began with creating a new directory and initializing a Git repository, followed by setting up a Go module and creating a main.go file. We wrote a simple "Hello, World!" program to verify that our setup was functioning correctly. After implementing a feature that displays a custom message when visiting the root URL, we committed our changes, switched branches, and merged the feature back into the main branch. Finally, we created a remote GitHub repository and pushed our local code, reinforcing our skills in version control and web application development.

                
                    package main

                import (
                    "log"
                    "net/http"
                )
                
                func home(w http.ResponseWriter, r *http.Request) {
                    w.Write([]byte("Welcome to my Go server!"))
                }
                
                func main() {
                    //mux
                    mux := http.NewServeMux()
                    mux.HandleFunc("/", home)
                    log.Println("Server is running on http://localhost:4000")
                    err := http.ListenAndServe(":4000", mux)
                    log.Fatal(err)
                }
