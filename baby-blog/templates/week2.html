Week 2

In Week 2, we focused on essential web development concepts and tools that are critical for both backend and frontend development. We explored various developer tools, including Linux, Git, Node.js, Go, VSCode, and TypeScript, each playing a role in setting up an efficient development environment. We also delved into the JavaScript execution model, understanding the call stack and event loop, and how asynchronous operations impact application performance. We learned about HTTP, web application architecture, and Node.js for server-side development, emphasizing the importance of structured architecture and asynchronous programming. To reinforce our learning, we initiated a small project involving setting up a web server and making HTTP requests.

                
                    package main

                import (
                    "log"
                    "net/http"
                )
                
                func home(w http.ResponseWriter, r *http.Request) {
                    w.Write([]byte("Welcome to my Go server!"))
                }
                
                func main() {
                    //mux
                    mux := http.NewServeMux()
                    mux.HandleFunc("/", home)
                    log.Println("Server is running on http://localhost:4000")
                    err := http.ListenAndServe(":4000", mux)
                    log.Fatal(err)
                }
